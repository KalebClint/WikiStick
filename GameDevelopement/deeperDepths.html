Overview

Deeper Depths is basically Doodle Jump, the main difference is that you're going down instead of up. You have to carefully descend the platforms, making sure not to fall too far, lest you get hurt. Different types of platforms, enemies and powerups make the descent more challenging.

Deeper Depths is the first 2d game I made, its an endless game where you try to descend as far down as possible, I first made a basic version of it for a game jam, but then liked it enough that I continued and made it better. I made it into a mobile game and implemented plenty of functionality to keep it interesting. I did upload the game to Google Play, however it is a little buggy and won't connect to play services, so it wont save or work with achievements. Maybe I'll fix this someday. We'll see.
